# Bayesian Decision Modeling
This repository contains MATLAB implementations for model evaluation in a Bayesian decision-making framework.

## Contents

- `model_evaluation_aicc.m`  
  Computes AICc values for candidate models and compares their predictive performance.

- `evaluate_models.m`  
  Function for calculating AICc from negative log-likelihood values.

## Methods

- Maximum Likelihood Estimation (MLE)
- Akaike Information Criterion corrected (AICc)

## Notes
